== Description ==
Make a SEO friendly URLs, clean URLs, permalinks, etc. You know, those nice-looking URLs that are easy to read (for humans and search engines).
Support: Vietnamese, French, Italian, Swedish, etc..

Inserting a permalink is easy using the [supermalink] shortcode:
* Example 1: [supermalink]This is a link of current post[/supermalink]
* Example 2: [supermalink id=150]This is a link to post 150[/supermalink]
* Example 3: [supermalink id=123 rel="post" class="your_class" target="_blank"]Open post 123 in a new window[/supermalink]

== Installation ==
1. Upload Supermalink's folder to the `/wp-content/plugins/supermalink` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
